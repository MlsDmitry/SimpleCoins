## SimpleCoins Commands

| Default command | Parameter | Description | Default Permission |
| :-----: | :-------: | :---------: | :-------: |
| /coins | | Shows your money | All |
| /topcoins WIP | | Shows top money | All |
| /addcoins | `<coins>` | Adds `<money>` To a player | `OP` |
